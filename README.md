# Ricing BSPWM

#### Config files for:

- `alacritty`
- `bspwm`
- `mpd`
- `nvim`
- `polybar`
- `rofi`


Base theme and inspiration credits to polybar-themes forest (https://github.com/adi1090x/polybar-themes)

### Fonts

Here's a list of all fonts used by this theme.

**`Text Fonts`**
- Iosevka Nerd Font
- Fantasque Sans Mono
- Noto Sans
- Droid Sans
- Terminus

**`Icon Fonts`**
- Iosevka Nerd Font
- Icomoon Feather
- Material Icons
- Waffle (Siji)

### Installation

- Clone this repository -
```
$ git clone --depth=1 https://github.com/AlessandroPerazzetta/ricing-bspwm.git
```

- Change to cloned directory and make setup.sh executable -
```
$ cd ricing-bspwm
$ chmod +x setup.sh
```

- Run `setup.sh` -
```
$ ./setup.sh
